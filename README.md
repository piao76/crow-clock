稍微寫一下撞到的坑給自己紀錄
1. 應該要先梳理好邏輯規則再實作，這次一直失敗是因為除了大時段以外還有小時段，小時段的權重比例不同，白天與夜晚的時間長度也不同。(太小看曆法了自己果然還是要先熟悉內容)
2. 因為是以日出日落時間作為判斷，所以多少還是會有誤差，就算撈了api來協助，但每個網站抓的api可能不同，目前還是有兩分鐘的誤差。
3. 樣式很容易在修改邏輯的過程中跑掉。
4. 看來還要加明暗月的判斷。
